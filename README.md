# Super Mario Side-Scroller

A Mario-themed side-scroller utilizing Flappy Bird mechanics, built with the frontend Holy Trinity — HTML, CSS, and JavaScript. No frameworks. No dependencies. Just the web.

---

## Gameplay

Guide Mario through an endless run of incoming pipes. Press any key or tap the screen to jump — time it right or it's game over. The longer you survive, the better.

- **Jump** — `Spacebar` / `Tap` (mobile)
- **Restart** — Click the on-screen button after a collision or press spacebar

---

## Built With

| Technology | Role |
|------------|------|
| HTML5 | Game structure and asset layout |
| CSS3 | Animations, keyframes, and responsive design |
| JavaScript | Game loop, collision detection, and state management |

## Getting Started

No build tools or installs required. Just clone and open.

```bash
git clone https://github.com/Mister-EF/Jogo-HTML-CSS-JS.git
cd super-mario-side-scroller
```

Then open `index.html` in your browser — that's it.

---

## Features

- Smooth CSS keyframe animations for Mario, pipes, and clouds
- Collision detection via a continuous JavaScript game loop
- Game-over state with overlay and full restart support
- Touch event support for mobile play
- Responsive layout adjustments for smaller screens
